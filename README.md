# pyrrhogaster-image-analyzer
A gift for my "senpai"

## Usage
- あるフォルダ(ここでは`source`)に画像をまとめて入れる
- 画像の識別子と、その画像のパスをCSVファイル(ここでは`index.csv`とする)に記述
- ファイルエンコーディングやファイル名を`main.py`の上の方にある`User Configuration`部分で設定する

- `main.py`を実行
```
python main.py
```

## Requirements
- Python 3.8
